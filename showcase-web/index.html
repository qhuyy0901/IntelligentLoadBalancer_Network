Intelligent Load Balancer — University Networking Project

Project Overview

A showcase web dashboard for the Intelligent Load Balancer university project. Demonstrates real-time server monitoring, traffic distribution, and health-check visualisation for an AWS EC2-based load balancing system.

HTML / CSS / JS AWS EC2 Load Balancer Health Check

Disclaimer

All data is simulated in-memory with JavaScript arrays. There is no backend and no database. Everything resets on page reload. This is a UI/UX demo designed to showcase system architecture understanding.
