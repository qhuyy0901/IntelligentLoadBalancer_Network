About This Project

This is a showcase web dashboard built for the Intelligent Load Balancer university project. It demonstrates real-time server monitoring, traffic distribution, and health-check visualization for an AWS-based load balancing system.

HTML / CSS / JS AWS EC2 Load Balancer Health Check

Disclaimer

All data displayed on this dashboard is simulated in-memory using JavaScript arrays. There is no backend server or database involved. Data resets on every page reload. This demo is designed to showcase UI/UX concepts and system architecture understanding.
